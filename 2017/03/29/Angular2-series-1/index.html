Angular2（一）

发表于 2017-03-29 | 分类于 关于Angular2 | | 阅读次数
本文主要介绍Angular-Cli的相关配置和基础用法



###前言

本文所有内容均是参考大漠穷秋老师的Angular2.0视频教程,你可以点击上方链接快速进入视频教程版，下面的内容纯粹只是个人记录，怕自己忘了。

###一

使用 npm install -g angular-cli 安装angular-cli
PS: 这里大漠穷秋老师不推荐使用cnpm安装，因为会导致部分插件目录不一致

angular-cli它集成了很多工具，比如：karma、webpack、typescript等等

###二

1.使用 ng new 项目名称 初始化一个项目;
2.项目初始化完成后切换到项目文件目录;
3.在项目目录下使用 npm isntall 安装package.json文件中的依赖
4.等待安装完成

###三

使用 ng serve 启动项目

PS: 使用 npm serve –
